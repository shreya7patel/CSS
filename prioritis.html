Priorities

Types of CSS

  • Inline CSS
  • Internal CSS
  • External CSS

What is Priorities Of CSS Types ?

  1. Inline CSS Has the Priorities at first number
  2. Internal CSS Has the Priorities at Second number
  3. External CSS Has the Priorities at Third number

Example :

Lorem ipsum dolor sit, amet consectetur adipisicing elit. Nobis eaque animi dolorem id doloribus pariatur praesentium voluptatum, reprehenderit quod molestias natus aspernatur tenetur eveniet, perferendis porro! Est alias unde deleniti?

- Here we have written p tag and applied css with various types

  1. WITH INLINE CSS : 
    <p style="color:brown">lorem</p>

  2. WITH INTERNAL CSS :

    <style>
    p
    {
      color:brown;
    }
    </style>

  3. WITH EXTERNAL CSS :

    p
    {
      color:brown;
    }

NOTE : in the above example the Priorities of inline css will be consider as first then after internal css and then after External css
